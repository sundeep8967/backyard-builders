# User Roles and Actors

**Date:** 2025-11-14 || 19:49

The platform is designed to serve a distinct set of roles, each with specific responsibilities.

- **Customer:** The end-user homeowner.
- **System Admin (The "Factory Engineer"):** An internal superuser operating the separate, secure `admin-portal` to manage the core product catalog and pricing logic.
- **In-App Admin (The "Dealership Manager"):** The highest-level user *within the main application*, responsible for daily business operations, people management, and final approval on financial matters.
- **Customer Service Manager:** A support-focused user with read-only access to most data but full access to communication tools.
- **Design Advisor:** A sales-focused role managing leads and creating final contracts.
- **Project Manager:** An operations-focused role managing active builds post-contract.
- **Accountant:** A finance-focused role managing all billing, payments, and financial compliance.
- **Field Crew:** A mobile-focused, view-only role for on-site staff.
- **Warranty Coordinator:** A post-project support role for managing warranty claims.
